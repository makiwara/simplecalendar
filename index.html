The simplecalendar plugin enhances datetime inputs:

Usage

The simplecalendar is a jQuery plugin which is used in a conventional way: 

$('input.datetime').simplecalendar();
$('input#byid').simplecalendar();

When user clicks on designated input, a calendar pop-up would be displayed nearby. The input would be popuplated with selected datetime in dd-mm-yy format; let us look at the sample below.

Configuration options

Plugin provides a lot of options to modify its behavior: starting from pop-up height, disabled days and holidays and up to language and i18n customisation.

Let us hope the following code sample with default values gives an overview of customisation capabilites of the plugin. 

var defaults = {
    prerender: true, 
    visible: false, 
    prevWeeks: 2,   
    nextDays: 180, 
    rows: 8, 
    disabledDays: [6], // 0–Monday...6–Sunday
    disabledDates: ['11-03-2015'], //  Dates or 'dd-mm-yyyy'
    anchorDate: false, 
    languagePackage: {
        months: ['JANUARY', 'FEBRUARY', 'MARCH', 'APRIL', 
                 'MAY', 'JUNE', 'JULY', 'AUGUST', 'SEMPTEMBER', 
                 'OCTOBER', 'NOVEMBER', 'DECEMBER'],
        weekdays: ['Mo', 'Tu', 'We', 'Th', 'Fr', 'Sa', 'Su'], 
        startsSunday: false
    },
    offset: { 
        left: 100,   // px or % of input width
        top: '50%',  // px or % of input height
        shift: '50%' // px or % of calendar height
    },
    cssPrefix: 'simplecalendar'
};
